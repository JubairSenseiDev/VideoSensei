## 11. Responsive Behavior

### Mobile (Android, < 600dp)
- Single-column layouts throughout
- Bottom navigation OR hamburger drawer (decide in Phase 3)
- Cards full-width
- Presets wrap to 2Г—3 grid

### Tablet / Desktop (вүҘ 600dp)
- Two-column layouts where appropriate
- Side navigation rail
- Cards in grid (2-3 per row)
- Presets in single row of 5

### Desktop (Linux/Windows)
- Window min size: 800Г—600
- Default size: 1100Г—750
- Drag-and-drop zones highlighted
- Keyboard shortcuts: Ctrl+O (open), Ctrl+B (batch), Ctrl+, (settings)

## 12. Motion Design Principles

- **Hero transitions** between file selection вҶ’ configure вҶ’ result
- **Staggered fade-in** for list items (50ms per item)
- **Spring physics** for cards (not linear)
- **Progress ring** animates with curve `Curves.easeInOutCubic`
- **Confetti** on result screen (60 particles, 1.5s, gravity 0.4)
- **Shimmer loaders** for any async data > 200ms wait
- **Never** use linear easing for organic motion
- **Always** respect `MediaQuery.disableAnimations` (accessibility)

## 13. Dark vs Light

> рҹ“Ң **Full token reference: [`THEME.md`](../THEME.md)**

**Dark mode (DEFAULT)** вҖ” honors the bash-CLI heritage, terminal aesthetic:
- Background: `#0A0A0B` (near-black, slightly cool)
- Cards / elevated: `#111114`
- Primary text: `#FFFFFF`
- Secondary text: `#A1A1AA`
- Muted text: `#52525B`
- Accent: `#00FF88` рҹҹў **NEON GREEN** вҖ” the signature color
- Hairline borders: white at 8вҖ“12% opacity
- Progress ring glow: `0 0 24px rgba(0,255,136,0.4)`
- Glass surfaces: `backdrop-filter: blur(12px)` + white at 4% opacity

**Light mode** вҖ” warm, paper-like, accessible:
- Background: `#F0F0EC` (warm cream)
- Cards / elevated: `#FAF9F5`
- Primary text: `#1E1914` (warm near-black)
- Secondary text: `#50483C`
- Muted text: `#786E5F`
- Accent: `#008246` рҹҢІ **FOREST GREEN**
- Hairline borders: `#1E1914` at 10% opacity
- Glass surfaces: `#1E1914` at 4% opacity, `blur(12px)`

**Auto mode**: follows system setting (Android 10+, iOS 13+, Linux/Windows via Flutter platform channel).

### Why dark-first?

1. **Brand consistency** вҖ” jubairsensei.com is dark-first; VideoSensei inherits
2. **Terminal heritage** вҖ” the original VideoSensi was a bash CLI; green-on-black honors that
3. **Compression focus** вҖ” when you spend 5+ minutes watching a progress ring, dark mode is easier on the eyes
4. **Saved bytes = green** вҖ” the accent color reinforces success semantically

### Theme switch behavior

- Settings вҶ’ Theme: `[ Dark в–ҫ ]` (default), `Light`, `Auto`
- Switch is instant вҖ” no app restart
- Selection persists across sessions
- `Auto` mode re-evaluates when system theme changes
